# cs-335-website

Download this repo to your computer and click on the HTML file to traverse through the Bob Doran Museum of Computing website, created as a part of the COMPSCI 335 course at the University of Auckland.

This web application was created using HTML, CSS and Javascript. Information on each page of the web application is retrieved by sending GET/POST requests to the Bob Doran Museum of Computing APIs/Endpoints.

## The website has the following features:

1. Can view displays available at the museum.
2. Can search through articles available.
3. Can post comments on a Guestbook.
